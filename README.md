# ZMesh
ZMesh目前只是一套数据结构, 后续会添加可视化之类的功能和一些几何处理算法

# 目录
* core: mesh数据结构
* io: 读写文件
  * [x] ObjReader
  * [ ] ObjWriter
  * [ ] OffReader 
  * [ ] OffWriter
  * [ ] 其他3D文件的 Reader 和 Writer
* gl: OpenGL相关
* algo: 算法相关
